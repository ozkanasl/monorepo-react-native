
### First install lerna globally
- Run
```
npm i -g lerna
```

### Then install packages
- Run
```
yarn bootstrap
```

### Start Web Version
- Run 
```
yarn web
```

### Start Mobile Version
- Run
```
yarn mobile
```


#### Delete All Node Modules
- Run
```
yarn clean
```

#### Clean Cache, Delete All Node Modules and reinstall again
- Run
```
yarn reset
```
